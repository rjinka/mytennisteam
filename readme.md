# Tennis Rotating Tool

This project is a tool for managing tennis rotations. It consists of a backend server and a frontend client.

## Backend

The backend is a Node.js server using Express and MongoDB.

### Setup and Running

1. Navigate to the `backend` directory:
   ```sh
   cd backend
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the server:
   ```sh
   npm start
   ```
The server will be running on `http://localhost:3000`.

## Frontend

The frontend is a vanilla JavaScript application built with Vite.

### Setup and Running

1. Navigate to the `frontend` directory:
   ```sh
   cd frontend
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

The frontend will be running on `http://localhost:5000`.
